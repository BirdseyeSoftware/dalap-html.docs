dalap-html

Render HTML templates using dalap

Get Started! » Read doc guides » Contribute »

Clojure data structures

dalap-html relies heavily on the usage of Clojure data structures to represent HTML templates. Why learn another template language when you already know how to use Clojure code?

A single template, multiple ways to render

Once a template is defined doesn't mean you are stuck with that implementation. dalap-html allows you to change your template rendering later on by the usage of dalap rules selectors.

Clojurescript ready

This library works well for rendering templates both in Server and Client side.
